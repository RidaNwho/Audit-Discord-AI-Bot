Bot Setup Guide

Follow these steps to get your Discord Gemini AI Bot up and running.

1

Create a Discord Bot

First, you need to create a new Discord application and bot user:

  1. Go to the Discord Developer Portal
  2. Click on "New Application" and give it a name (e.g., "Gemini AI Bot")
  3. Navigate to the "Bot" tab in the left sidebar
  4. Click "Add Bot" and confirm
  5. Under the "Privileged Gateway Intents" section, enable:
    • Presence Intent
    • Server Members Intent
    • Message Content Intent (important for reading messages!)
  6. Save your changes
Important: The Message Content Intent is required for the bot to read and respond to messages in Discord.
2

Get Your Discord Bot Token

Now you need to get your Discord bot token:

  1. On the Bot tab, click "Reset Token" (or "Copy" if you already have a token)
  2. Copy the token that appears
  3. Go to your bot dashboard: API Keys Settings
  4. Paste the token in the "Discord API Token" field and save
Security Warning: Never share your bot token with anyone or commit it to public repositories. This token provides full access to your bot.
3

Invite Your Bot to Your Server

Let's create an invite link for your bot:

  1. Go to the "OAuth2" tab in the left sidebar of the Discord Developer Portal
  2. Select "URL Generator" under OAuth2
  3. In the "Scopes" section, select "bot" and "applications.commands"
  4. In the "Bot Permissions" section, select:
    • Read Messages/View Channels
    • Send Messages
    • Embed Links
    • Attach Files
    • Read Message History
    • Add Reactions
  5. Copy the generated URL at the bottom of the page
  6. Open this URL in your browser and select the Discord server where you want to add your bot
  7. Follow the prompts to authorize the bot
4

Get a Gemini API Key

To use Google's Gemini AI with your bot, you need a Gemini API key:

  1. Go to Google AI Studio
  2. Sign in with your Google account
  3. Create a new API key or use an existing one
  4. Copy the API key
  5. Go to your bot dashboard: API Keys Settings
  6. Paste the key in the "Gemini API Key" field and save
Note: Google may offer a free tier of usage for Gemini API with a limited number of requests per minute and per month. Check Google's pricing for more details.
5

Start Your Bot

Once you've configured both API keys, you can start your bot:

  1. Go to your bot dashboard: Dashboard
  2. Click the "Start Bot" button in the Bot Status section
  3. Verify that the bot status changes to "Online"
  4. Check your Discord server - your bot should now be online!
6

Test Your Bot

Let's test your bot to make sure it's working properly:

  1. In your Discord server, type !ask What is artificial intelligence?
  2. The bot should respond with an explanation of AI powered by Gemini
  3. To analyze an image, type !analyze and attach an image to your message
  4. For help with available commands, type !help_gemini
Congratulations! If your bot responded, you've successfully set up your Discord Gemini AI Bot!
7

Customize Your Bot

Now that your bot is working, you can customize it:

  • Add custom commands - Create commands with predefined prompts or templates
  • Adjust configuration - Change the command prefix, response temperature, etc.
  • Read the documentation - Learn about all the features and options available

Need Help?

If you encounter any issues setting up your bot, check these common troubleshooting steps:

  • Verify that both API keys are correctly entered
  • Ensure the bot has the Message Content Intent enabled
  • Check that the bot has the necessary permissions in your Discord server
  • Look at the logs in the Logs section for any error messages

For more assistance, visit the project repository or create an issue if you encounter a bug.
